About Node.js

Node предназначен для создания масштабируемых сетевых приложений как среда выполнения JavaScript с асинхронным управлением событиями. В следующем примере «hello world» многие соединения могут обрабатываться одновременно. При каждом соединении вызывается обратный вызов, но если не нужно выполнять какую-либо работу, узел будет спать.
